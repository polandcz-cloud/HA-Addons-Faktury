# FakturaApp - Home Assistant Addon

Český fakturační systém přímo v Home Assistantu.

## Instalace

1. V Home Assistant jděte do **Nastavení → Doplňky → Obchod s doplňky**
2. Klikněte **⋮** → **Repozitáře** → přidejte: `https://github.com/polandcz-cloud/HA-Addons-Faktury`
3. Najděte **FakturaApp** a nainstalujte
4. Zapněte **"Zobrazit v postranním panelu"**

## Funkce

- Faktury s ARES lookup (IČO → automatické vyplnění údajů)
- Cenové nabídky s workflow (rozpracovaná → odeslaná → schválená → faktura)
- QR Platba na fakturách
- Rozpracované faktury (průběžné přidávání položek)
- Pravidelné/opakující se faktury
- Plátce i neplátce DPH
- Logo firmy (obrázek nebo text)
- PDF export
- Dashboard s přehledy
- Databáze odběratelů
- Automatické zálohy
